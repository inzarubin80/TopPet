# Иконки приложения

Создана SVG иконка `icon.svg` с изображением питомца в короне для конкурсов красоты животных.

## Текущий статус

✅ SVG иконка создана и используется как основной favicon (поддерживается современными браузерами)
✅ Название приложения изменено на "Top-Pet" в index.html и manifest.json

## Для полной совместимости (опционально)

Для поддержки старых браузеров и PWA рекомендуется конвертировать `icon.svg` в следующие форматы:

1. **favicon.ico** - многоразмерный ICO файл (16x16, 32x32, 48x48)
2. **logo192.png** - PNG 192x192 пикселей  
3. **logo512.png** - PNG 512x512 пикселей

## Онлайн инструменты для конвертации:

**Рекомендуется:** https://realfavicongenerator.net/
- Загрузите `icon.svg`
- Получите все необходимые форматы автоматически
- Скопируйте созданные файлы в папку `public/`

**Альтернативы:**
- https://favicon.io/ - конвертация SVG в favicon
- https://svg-to-ico.org/ - конвертация SVG в ICO
- https://ilovefavicon.com/converter - универсальный конвертер

## Или используйте командную строку (если установлен ImageMagick):

```bash
cd client/public

# Создать PNG файлы
convert icon.svg -resize 192x192 logo192.png
convert icon.svg -resize 512x512 logo512.png

# Создать ICO файл (требует специальных инструментов)
# Используйте онлайн конвертер для ICO
```

**Примечание:** SVG favicon уже работает в современных браузерах, поэтому конвертация не обязательна для базовой функциональности.
